

# Amorgens server



## Features

Users can 
- Authenticate 
- Create account 
- login and get token


Buyers can: 
- Create buy order
- Confirm buy order 
- View buy order
- report buy order
- cancel buy order


Seller can: 
- create sell order
- view sell order
- view buy orders under sell order
- confirm sell order
- cancel sell order
